The backend server runs on 127.0.0.1:8000

To access the admin panel to create Users, Listings, etc. manually,
go to 127.0.0.1:8000/admin/

Note: if VS CODE does not recognize "rest_framework" and you
are in a virtual Python environment, make sure that the Python
interpreter in the venv folder is selected

--- API ENDPOINTS ---

127.0.0.1:8000/listings/
    GET -> returns all listings
    POST -> add new listing

127.0.0.1:8000/listings/<username>/
    GET -> returns all listings belong to <username>

127.0.0.1:8000/listings/<username>/<title>/
    PUT -> updates listing
    DELETE -> removes listing

127.0.0.1:8000/wishlist/<username>/
    GET -> returns all listings in <username>'s wishlist
    POST -> add new wishlist listing for <username>

127.0.0.1:8000/users/
    GET -> returns all users
    POST -> add new user
